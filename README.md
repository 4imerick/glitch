# **Glitch**

The Glitch project is a simple CSS animation of a glitch effect.

## **Setup**

- HTML 5
- CSS 3

To use the projet follows these steps:

- Git clone the `dev` branch of the repository
- Go to the `template` folder
- Open the `index.html` file
